# 콘솔 게시판


## 도전

1. 요구사항에 맞게 **구현**

2. 객체 지향 프로그래밍 **OOP** 스럽게 최대한 적용해보기.

<br>

## 기능

### 메뉴

```bash
==============================================================
메뉴
==============================================================
1. 전체 글 조회 2. 글 번호로 조회 3. 글 등록 4. 글 삭제 5. 종료
==============================================================
> 
```

### 전체 글 조회

```bash
==============================================================
메뉴
==============================================================
1. 전체 글 조회 2. 글 번호로 조회 3. 글 등록 4. 글 삭제 5. 종료
==============================================================
> 1
==============================================================
번호                        제목                      작성자
==============================================================
1            안녕하세요                               홍길동
2            반갑습니다                               김첨지
==============================================================
메뉴
==============================================================
1. 전체 글 조회 2. 글 번호로 조회 3. 글 등록 4. 글 삭제 5. 종료
==============================================================
>
```

### 글 번호로 조회

```bash
==============================================================
메뉴

==============================================================
> 2
==============================================================
번호 입력 > 2
==============================================================
번호 : 2
작성자 : 김첨지
제목 : 반갑습니다.
본문 : 만나서 반가워요
==============================================================
메뉴
==============================================================
1. 전체 글 조회 2. 글 번호로 조회 3. 글 등록 4. 글 삭제 5. 종료
==============================================================
>
```
